# 🏫 Smart Campus Management System (DSA Project)

## 📌 Overview

The Smart Campus Management System is a console-based C++ application developed as a Data Structures and Algorithms (DSA) project. It simulates various campus management operations by implementing multiple data structures and algorithms.

## 🚀 Features

### 📍 Graph Module
- Dijkstra's Shortest Path Algorithm
- Breadth First Search (BFS)
- Depth First Search (DFS)
- Campus Navigation

### 🌳 Binary Search Tree (BST)
- Add Locations
- Search Locations
- Delete Locations
- Display Locations

### #️⃣ Hash Table
- Fast Location Lookup
- Collision Handling using Chaining

### 📥 Queue
- Request Management
- Process Requests

### 📚 Stack
- Search History
- Undo Recent Searches

### 🔍 Searching Algorithms
- Binary Search
- Linear Search

### 📊 Sorting Algorithms
- Merge Sort
- Bubble Sort
